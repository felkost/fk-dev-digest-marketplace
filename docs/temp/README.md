permanent project documentation. It exists so research doesn't get lost mid-build; it is **not**
linked from the root `README.md`, `CONTRIBUTING.md`, or any `docs/*.md` file, and it must not
become one by accident.

for what "released" means here — first tags cut for all four plugins). Track this as a to-do; it
is not automated.
