# Виправлення поширеної шпаргалки «Statistics Cheatsheet for Data Science»

Ця шпаргалка широко шириться LinkedIn і містить грубі помилки — ймовірно, це
AI-генерована картинка з «формулоподібним» текстом. Якщо користувач приносить
формули з неї, звіряйте тут. Авторитетне джерело для кожного розподілу —
його картка на чарті UDR (`leemis-navigation.md`).

## Помилки в формулах

| У шпаргалці (хибно) | Правильно | Джерело |
|---|---|---|
| `P(X=k) = (1)p²(1−p)⁻ᵏ/(n−β)` для Binomial | `P(X=k) = C(n,k) pᵏ (1−p)ⁿ⁻ᵏ` | `Binomial.pdf` |
| Normal: `P = (1/√(2πσ)) ẽ^(−u²)` без σ² і з довільним u | `f(x) = 1/(σ√(2π)) · exp(−(x−µ)²/(2σ²))` | `Normal.pdf` |

| `R² = SSR = sum of errors + t·SSR/SSL` | `R² = 1 − SS_res/SS_tot` (еквівалентно `SS_reg/SS_tot` для OLS з константою) | — |
| Кореляція `r = Σxᵢ−x̂ᵢȳ/√(Σ+y²)` | `r = Σ(xᵢ−x̄)(yᵢ−ȳ) / √(Σ(xᵢ−x̄)² Σ(yᵢ−ȳ)²)` | — |

| ЦГТ: `X ~ σ appx N(µ, e^{2/}...)` (нечитабельне) | для iid зі скінченною σ²: `X̄ ⟶ N(µ, σ²/n)` при n→∞ | контрприклад без σ² — Cauchy, див. SKILL «Режими відмови» |

## Концептуальна помилка, найгірша з усіх

Шпаргалка ставить в один стовпчик:

> Empirical rule ~68% within 1σ · **P-value ~95% within 2σ** ·
> **Significance Level α ~99.7% within 3σ**

Це змішує три різні речі:

- **Емпіричне правило 68–95–99.7** — властивість нормального розподілу
  (частки маси в межах 1σ/2σ/3σ). І тільки його.
- **p-value** — ймовірність, за істинності H₀, отримати статистику принаймні
  таку екстремальну, як спостережена. Це функція від даних, а не константа
  «95%».
- **Рівень значущості α** — обраний ДО експерименту поріг для p-value
  (типово 0.05). Це рішення дослідника, а не властивість розподілу.

Зв'язок між ними лише один: для двобічного z-тесту критичне значення при
α=0.05 дорівнює 1.96σ ≈ 2σ — звідси й плутанина.

## Що в шпаргалці прийнятне

Медіана/мода описово; means/variance для Binomial (np, np(1−p)) і Poisson (λ);
IQR = Q₃−Q₁ (у шпаргалці Q₂−Q₁ — теж помилка, це не IQR); z-score
`z = (x−µ)/σ`; означення викидів як «спостережень далеко від решти» — прийнятне
як інтуїція, робоче правило — 1.5·IQR або аналіз хвоста.
