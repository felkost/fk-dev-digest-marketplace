## Кістяк SARSA-сімейства (сигнатура нотбука)

```python
import numpy as np
from collections import defaultdict

def epsilon_greedy(Q, state, nA, eps, rng):
    if rng.random() > eps:
        return int(np.argmax(Q[state]))
    return int(rng.integers(nA))

def td_target(Q, s_next, a_next, reward, gamma, method, eps, nA):
    if method == "SARSA":            # on-policy: фактична наступна дія
        nxt = Q[s_next][a_next]
    elif method == "SARSAmax":       # Q-learning, off-policy
        nxt = np.max(Q[s_next])
    elif method == "ESARSA":         # Expected SARSA: матсподівання по політиці
        p = np.full(nA, eps / nA); p[np.argmax(Q[s_next])] += 1 - eps
        nxt = float(p @ Q[s_next])
    return reward + gamma * nxt

def train(env, num_episodes=500, alpha=0.1, gamma=0.9, eps=1.0,
          eps_decrease="decay", eps_decay=0.9, eps_min=0.01,
          method="SARSA", seed=1):
    rng = np.random.default_rng(seed)
    nA = env.action_space.n
    Q = defaultdict(lambda: np.zeros(nA))
    returns = []
    for i in range(1, num_episodes + 1):
        if eps_decrease == "1/n":
            eps = 1.0 / i
        elif eps_decrease == "decay":
            eps = max(eps * eps_decay, eps_min)
        s, _ = env.reset(seed=int(rng.integers(1e9)))
        a = epsilon_greedy(Q, s, nA, eps, rng)
        done, G = False, 0.0
        while not done:
            s2, r, term, trunc, _ = env.step(a)
            done = term or trunc
            a2 = epsilon_greedy(Q, s2, nA, eps, rng) if not done else 0
            target = td_target(Q, s2, a2, r, gamma, method, eps, nA)
            Q[s][a] += alpha * (target - Q[s][a])
            s, a, G = s2, a2, G + r
        returns.append(G)
    return Q, returns
```

Значення нотбука: `alpha=.1, gamma=.9, eps=.1` (константний базлайн) і розклади
`eps_decay=0.95/0.9`, `eps_min=0.2/0.01`, `alpha_decay=0.9, alpha_min=0.1`,
500 епізодів.

## gymnasium (опційно)

```python
# pip install gymnasium
import gymnasium as gym
env = gym.make("FrozenLake-v1", is_slippery=False)   # нотбук: детермінований старт
env = gym.make("FrozenLake-v1", is_slippery=True)    # потім стохастика: eps_min>0!
```

API 2026: `reset()` повертає `(obs, info)`; `step()` —
`(obs, reward, terminated, truncated, info)`; done = `terminated or truncated`
(стара пара `done` з gym<0.26 — типова помилка старих прикладів).
Reward shaping — через `gymnasium.RewardWrapper` (нотбук має приклад); це зміна
задачі, документувати.

## Пастки

| Пастка | Захист |
|---|---|
| `done` замість `terminated or truncated` | сучасний API gymnasium |
| порівняння методів із різними ε | вирівняти розклади і seed |
| оцінка політики з ε навчання | окремий greedy-прогін (eps≈0.01) |
| ε згасло до 0 у стохастичному середовищі | `eps_min > 0` |
| γ під «стабільність» | γ — з горизонту задачі; стабільність — через α |
| `Q[state]` на неперервних станах | дискретизація або DQN, таблиця не масштабується |
